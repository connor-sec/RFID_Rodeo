# RFID_Rodeo
RFID Rodeo — offline browser tool for access-control card formats. Encode/decode Wiegand &amp; smart-card credentials, compare, convert, auto-detect captures, and generate Proxmark clone commands &amp; Flipper .rfid files. Layouts &amp; parity verified against the Proxmark3 (Iceman) reference. Authorized research &amp; education only.
